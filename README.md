O desafio consiste no desenvolvimento de uma aplicação interativa que utilize a estrutura de dados grafo
para resolver um problema inspirado em uma situação real. A proposta é que o grupo modele o problema como
um grafo, implemente essa estrutura por meio de listas ou matrizes de adjacência e desenvolva funcionalidades
que permitam criar, modificar e consultar o grafo, além de aplicar algoritmos relevantes à situação modelada.
